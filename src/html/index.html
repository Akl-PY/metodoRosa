Sobre el Método ROSA

La evaluación ergonómica mediante el Método ROSA (Rapid Office Strain Assessment) permite analizar y evaluar el entorno de trabajo, identificando y reduciendo riesgos ergonómicos para mejorar el bienestar y prevenir lesiones.

    En esta página, podrás:

  • Completar una evaluación que considera factores como tiempo de uso diario, altura del asiento y ajustes adicionales del mobiliario.
  • Acceder a descripciones e imágenes para comprender mejor cada criterio de evaluación.
  • Recibir recomendaciones para mejorar la ergonomía en tu espacio de trabajo.
